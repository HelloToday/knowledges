#### 为什么要用https?

回答：因为http采用的是明文传播 很容易被拦截，并且获得你传输的数据。

#### https的安全性主要体现在哪儿？

HTTPS相对HTTP提供了更安全的数据传输保障，主要体现在三个方面：

* 1）内容加密：客户端到服务器的内容都是以加密形式传输，中间者无法直接查看明文内容；

* 2）身份认证：通过校验保证客户端访问的是自己的服务器；

* 3）数据完整性：防止内容被第三方冒充或者篡改。
